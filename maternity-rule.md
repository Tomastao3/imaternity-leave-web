# 产假规则模板

## 示例数据
| 城市 | 产假类型 | 产假天数 | 是否顺延 | 是否包含工作日 |
|------|----------|----------|----------|----------------|
| 北京 | 法定产假 | 98       | 是       | 是              |
| 上海 | 难产假   | 15       | 否       | 是              |

---

## 填写说明

| 字段名     | 说明                           | 是否必填 |
| ------- | ---------------------------- | ---- |
| 城市      | 城市名称，如：北京、上海、深圳              | 是    |
| 产假类型    | 法定产假 / 难产假 / 多胞胎 / 奖励假 / 流产假 | 是    |
| 产假天数    | 产假天数，必须为正整数                  | 是    |
| 是否顺延    | 是 / 否                        | 是    |
| 是否包含工作日 | 是 / 否                        | 是    |
