## 🔄 修改内容

### 1. MainLayout.js - 添加头部

```javascript
import MaternityCelebrationHeader from './MaternityCelebrationHeader';

const MainLayout = ({ session, onLogout }) => {
  return (
    <div className="main-layout">
      <Sidebar ... />
      <div className="main-content">
```

## 🔧 技术实现

### 1. 头部固定
```css
position: sticky;
top: 0;
z-index: 100;
```
- 滚动时头部保持在顶部
- z-index确保在其他内容之上

### 2. 弹性布局
```css
display: flex;
flex-direction: column;
```
- 头部和内容垂直排列
- 内容区域自动填充剩余空间

### 3. 退出功能
```javascript
<MaternityCelebrationHeader onLogout={onLogout} />
```
- onLogout从App传递到MainLayout
- 点击退出按钮触发登出
